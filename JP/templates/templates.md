# 業務テンプレート（アプリのたたき台）

「在庫管理」「営業支援 (SFA)」「プロジェクト管理」の 3 つは、**そのままでも業務で使える完成形のアプリ** をテンプレートとして同梱したものです。デザイナで新規プロジェクトを作るときに選ぶだけで、画面・データモデル・サンプルデータが一式展開されます。

<img src="../../Image/web/patterns/create_new.png" alt="新規プロジェクト作成ダイアログのテンプレート選択" style="border: 1px solid #ccc;" width="500">

## アプリ作成パターン集との位置づけ

| | [アプリ作成パターン集](../patterns/patterns.md) | 業務テンプレート（このページ） |
|---|---|---|
| 単位 | **部品** (検索 / リスト / レイアウト 等) | **アプリまるごと** (画面 + データモデル + サンプルデータ) |
| 目的 | 「この場面はどう作る？」を引く | **コピーして自社向けに改造する起点** |
| 内容 | パターンごとに最小サンプル | 業務として成立する画面構成・ダッシュボード・サンプルデータ |
| 使い方 | マニュアルで読んで該当モジュールを真似る | デザイナで新規プロジェクト → そのまま編集 |

業務テンプレートは **「明日から使えるアプリのたたき台」** が目的です。項目追加・画面変更はすべてデザイナ GUI で行えるので、自社の用語・運用に合わせて改造しながら本番運用へ持っていけます。

---

## 一覧

| テンプレート | 内部名 | 概要 |
|---|---|---|
| [在庫管理テンプレート](inventory_management.md) | `InventoryManagementTemplate` | 入出庫・棚卸し・発注・在庫一覧・要発注品アラート・ダッシュボード |
| [営業支援 (SFA) テンプレート](sfa.md) | `SFATemplate` | 商談・活動履歴・顧客/担当者管理・パイプラインダッシュボード |
| [プロジェクト管理テンプレート](project_management.md) | `ProjectManagementTemplate` | プロジェクト・タスク・ガント・タスクボード・進捗ダッシュボード |

いずれも次の共通構成です。

- **DB**: SQLite サンプル同梱（PostgreSQL / SQL Server / MySQL / Oracle に差し替え可能）
- **実行形態**: Blazor WebAssembly
- **拡張ライブラリ**: ApexCharts（ダッシュボードのグラフ）。プロジェクト管理はさらに Codeer.LowCode.Blazor.Extras（ガント・タスクボード）を使用

---

## 共通の使い始め方

1. デザイナで「**新規プロジェクト**」 → テンプレート選択で目的のテンプレートを選ぶ
2. プロジェクトが展開され、サンプルデータ入りの SQLite DB がそのまま使える状態で起動できる
3. 触ってみて運用イメージを掴んだら、項目名・画面・マスタデータを自社向けに置き換える
4. 本番 DB を作る場合は `DB/` 配下の DDL を別 DB へ流し、`designer.settings.json` の DataSource を差し替える
