# 入手とライセンス

## 1. Visual Studio 拡張機能をインストール

Codeer.LowCode.Blazor はプロジェクトテンプレートとして Visual Studio Marketplace で配布しています。

- [Codeer.LowCode.Blazor.Templates（Visual Studio Marketplace）](https://marketplace.visualstudio.com/items?itemName=Codeer.LowCodeBlazor)

Visual Studio の拡張機能管理画面から直接インストールすることもできます。

## 2. トライアル（30 日間無料）

インストール後はすぐにトライアルとして使い始められます。

- **30 日間、無料で全機能を使用可能**
- 評価用途のみ
- 商用利用・商用製品の開発プロセスでの利用にはライセンス購入が必要

## ライセンス種別

用途に応じて以下のライセンスがあります。価格は[製品ページ](https://www.codeer.co.jp/LowCode#)をご覧ください。

### 開発時に必要

| ライセンス | 用途 | 利用可能台数 |
|---|---|---|
| **開発ライセンス** | 開発・テスト用。開発用 Web サーバーでも利用可 | 1 部署最大 30 台 |

### 運用時に必要

| ライセンス | 用途 | 制限 |
|---|---|---|
| **サーバーライセンス** | Web アプリの本番運用 | 1 サーバー／ 1 ドメインごと（ユーザー数非依存） |
| **クライアントライセンス** | WPF / WinForms のクライアントサーバー型アプリ | 最大 100 台 |
| **メンテナンスライセンス** | デザイナで設定変更のみ行うマシン用 | 1 台 |

マルチテナントなど特殊な利用形態は個別対応しますので、お気軽にご相談ください。

## ライセンス登録方法

開発・サーバーライセンスは複数の方法で登録できます。用途に応じて使い分けます。

### デザイナから登録

- [オンラインで登録](../overview/license_online_registration.md) / [解除](../overview/license_online_cancellation.md)
- [オフライン（Web 経由）で登録](../overview/license_web_registration.md) / [解除](../overview/license_web_cancellation.md)

### CLI ツールから登録（Windows）

- [Windows で CLI 登録](../overview/licence_windows_cli_registration.md)

### ドメインライセンス（クラウド環境向け）

マシン固有のキーが使えない PaaS などでは、ドメインをキーにして登録します。

- [ドメインライセンスの登録](../overview/domain_license_registration.md) / [解除](../overview/domain_license_cancellation.md)

### クライアントサーバーアプリ

WPF / WinForms のプロジェクトを作成するとライセンス登録用アプリも生成されます。

- [LicenseRegister について](../overview/license_license_register_application.md)

### Linux サーバー

- [Linux で CLI 登録](../overview/license_linux_registration.md)
